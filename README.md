## Installation

- Install OCaml.
- Create a copy of Config.default, name it Config and modify it to fit your configuration
- Run make


## Unit Tests

- Requires ocamlfind
- Requires ounit
- Install via OPAM (https://opam.ocaml.org)

``
	make TESTS
``
